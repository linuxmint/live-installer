Situada sobre os ombros de gigantes

  • Linux Mint é uma distribuição GNU / Linux baseada no Debian derivada e compatível com o Debian.
  • As nossas inovações, o rápido desenvolvimento do Debian e a quantidade enorme de pacotes Debian tornam o Linux Mint um dos sistemas operacionais mais atraentes para desktop disponível para usuários domésticos.
  • Nas casas, o Linux Mint é o 4º sistema operativo mais usado, atrás do Microsoft Windows, Apple Mac OS e Debian.
