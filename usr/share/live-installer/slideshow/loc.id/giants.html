Berdiri di pundak para raksasa.

  • Linux Mint adalah distro turunan dari sebuah GNU/Linux berbasis Debian dan kompatibel dengan Debian.
  • Inovasi-inovasi kami, pengembangan cepat Debian dan pemilihan paket Debian yang sangat banyak membuat Linux Mint menjadi salah satu sistem operasi desktop paling atraktif bagi pengguna rumahan.
  • Di rumah, Linux Mint adalah sistem operasi desktop paling banyak digunakan setelah Microsoft Windows, Apple Mac OS dan Debian.
